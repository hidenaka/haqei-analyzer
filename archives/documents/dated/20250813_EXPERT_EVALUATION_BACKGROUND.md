## 1. エグゼクティブサマリー

### 1.1 プロジェクト概要
HAQEIは、古代中国の易経（I-Ching）の64卦体系を現代的に解釈し、人格理解のメタファーとして活用する「仮想人格生成システム」です。占いや診断ではなく、創造的な自己表現ツールとして位置づけられています。

### 1.2 本資料の目的
アドバイス生成機能の品質評価にあたり、以下の専門的観点からのレビューを依頼するための背景情報を提供します：
- 心理学的アプローチの妥当性
- アルゴリズムの論理的整合性
- ユーザー体験の質
- 文化的配慮と倫理的側面

---

## 2. システムアーキテクチャ

### 2.1 Triple OS（三層人格モデル）
```
┌─────────────────────────────────────┐
│         Triple OS Framework          │
├───────────────┬───────────┬─────────┤
│  Engine OS    │Interface OS│Safe Mode│
│  (内的価値観) │ (社会性)   │  (防御) │
├───────────────┼───────────┼─────────┤
│  64卦から選択 │ 64卦から選択│64卦から │
│  質問1-12     │ 質問13-24  │質問25-36│
└───────────────┴───────────┴─────────┘
```

### 2.2 データフロー
1. **入力**: 36問の選択式質問（各4選択肢）
2. **処理**: 回答パターンから3つのOSを決定
3. **分析**: 512パターンの組み合わせ分析
4. **出力**: 個別化されたアドバイス生成

## 3. 理論的基盤

### 3.1 易経の現代的解釈
- **伝統的意味**: 占いや運命判断のツール
- **HAQEI的解釈**: 人間の多面性を表現する象徴体系
- **科学的立場**: 科学的根拠を主張せず、メタファーとして活用

### 3.2 心理学的考察
- **多面的自己理論**: 人格の状況依存的側面を3つのOSで表現
- **成長段階モデル**: 探索期→成長期→確立期→成熟期の4段階
- **バランス理論**: 内的一貫性と環境適応のバランス

## 5. 品質保証メカニズム

### 5.1 定量的品質指標
1. **一貫性スコア** (Consistency)
   - 回答パターンの内的整合性
   - 0-100%で数値化

2. **バランス指標** (Balance)
   - Triple OS間の調和度
   - 極端な偏りを検出

3. **カバレッジ** (Coverage)
   - 8次元（八卦）の網羅性
   - 多面的分析の完全性

### 5.2 定性的品質要素
- 文化的適切性の確保
- 判断的表現の回避
- 建設的・肯定的なトーン

## 6. 倫理的配慮

### 6.1 表現の原則
```
❌ 避けるべき表現:
「あなたは○○な人です」（断定的診断）

✅ 採用する表現:
「この仮想人格は○○が観察されます」（観察的記述）
```

### 6.2 免責事項
- 医学的・心理学的診断ではない
- エンターテインメント/自己探索ツール
- 個人の判断を最優先

## 7. 技術的実装

### 7.1 アーキテクチャ
- **言語**: JavaScript (ES6+)
- **フレームワーク**: Vanilla JS（依存性最小化）
- **データ構造**: 64卦×6爻 = 384パターン
- **UI/UX**: レスポンシブデザイン、アクセシビリティ配慮

### 7.2 アルゴリズムの特徴
- O(1)の高速パターンマッチング
- メモリ効率的なデータ構造
- クライアントサイド完結（プライバシー保護）
