🚀 回答データ一括処理（推奨）

複数人の回答データをまとめて貼り付けると、自動で解析・診断・結果生成まで一括実行します

✨ 貼り付けるだけで全自動処理

💡 使い方

  1. 上記のテキストエリアに、受け取った回答データをそのまま貼り付け
  2. 「🚀 一括処理実行」ボタンをクリック
  3. 自動で参加者登録 → 回答解析 → 診断実行 → 結果生成まで完了
  4. 完了後、各参加者の結果テキストをコピーしてユーザーに送信
