## 🎯 重要な注意点

1. **すべての色指定はCSS変数を使用**
   - 黒背景関連の色は使用しない
   - 白背景（`var(--white)`）をベースに設計

2. **グラデーションの使用**
   - `var(--gradient-primary)`: メインのアクセント
   - `var(--gradient-secondary)`: サブのアクセント

3. **シャドウの適切な使用**
   - カードや浮き上がる要素には必ずシャドウを付与
   - ホバー時はより強いシャドウに変更

4. **コントラストの確保**
   - 白背景に対して十分なコントラストを持つグレースケールを使用
   - `var(--gray-900)`: メインテキスト
   - `var(--gray-700)`: サブテキスト
   - `var(--gray-600)`: 補助テキスト

このCSSを使用することで、HAQEIの統一デザインシステムに完全に準拠した、美しく機能的なタブナビゲーションシステムが実装できます。
